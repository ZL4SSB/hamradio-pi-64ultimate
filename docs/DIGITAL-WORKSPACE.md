# Digital Workspace

Choose FT8, FT4, FT2, Q65, MSK144, FST4, FST4W or WSPR. Configure radio RX/TX
audio and CAT/PTT first. **Enable TX** arms transmission, **Auto sequence** allows
the future engine to progress a QSO, **HALT** stops TX, and **TUNE** is for level
setup into a dummy load.

**Load Preview** supplies labelled UI test messages. It does not decode RF. A
live proprietary-capable DSP/decoder provider is not included in this build.
