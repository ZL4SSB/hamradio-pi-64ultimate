# Managed External Applications

Programs in Applications remain separate executables. **Check Installed** scans
commands/packages, **Install** opens Raspberry Pi OS apt, **Launch** starts the
installed command, **Remove** delegates to apt or the registered Windows
uninstaller, and **Help** opens the program-specific HRPU guide.

External program licences apply independently. HRPU does not merge their source.
Automatic Windows installers are not supplied; use each publisher's supported
distribution and verify its licence and signature.
