# Hamlib Utilities

External radio/rotator command-line utilities. HRPU may use documented rigctld or
rotctld TCP interfaces as separate processes after distribution review. Configure
the exact model, serial device, baud rate and endpoint. Only HRPU should own the
physical CAT port; clients connect to the broker. Detection alone does not mean a
radio is connected. Hamlib's LGPL obligations apply to its distribution.
