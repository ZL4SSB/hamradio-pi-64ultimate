# Ultimate Shack Clock

Ultimate Shack Clock is built directly into HamRadio-Pi Ultimate.

It displays:

- local time and date
- UTC time and date
- station callsign and locator
- sunrise and sunset derived from the Maidenhead locator
- daylight/night status
- simplified greyline overview
- NOAA solar and geomagnetic conditions
- estimated HF band conditions
- placeholders for future weather, CAT radio status and DX information

No external HamClock service is required.

An optional external clock URL can still be configured under Preferences.
