# HamRadio-Pi Ultimate 1.0.0

Version 1.0 consolidates the modern single-window interface and the first
complete control-centre feature set.

## Included

- Live Raspberry Pi dashboard
- Application installation, launch, update, repair and removal
- USB, SDR, CAT, serial and audio hardware discovery
- WPSD download, drive inspection, backup and restore helpers
- NOAA SWPC propagation data
- Git repository update status
- Station settings
- System report generation
- Ham Radio desktop-menu grouping
- Desktop launcher and v1 installer
- Background command execution and live logs

## Safety

WPSD restore scripts never guess the target drive. The operator must enter the
device path twice before writing. No update is installed automatically.
