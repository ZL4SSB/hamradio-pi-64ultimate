# WSJT-X

External weak-signal program for modes including FT8 and FT4. Configure station
callsign/grid, rig control and one RX/TX audio device. When used with HRPU, connect
through a documented CAT broker endpoint; do not open HRPU's serial port twice.
Use the WSJT-X documentation for operating and frequency guidance. WSJT-X is GPL
software and remains a separate process; no source is incorporated into HRPU.
