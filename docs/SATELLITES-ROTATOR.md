# Satellites and Rotator

The intended chain is Satellite Engine → Rotator Control → Radio CAT Doppler
Correction. Live use requires a current TLE source, accurate UTC, Station Profile
locator, supported rotator, and CAT radio profile. AOS, LOS, azimuth, elevation
and Doppler must all identify their provider and age.

Preset positions are labelled preview geometry. Do not use them for antenna
movement or Doppler correction.
