# Desktop and menu shortcuts

## Windows

Double-click `Install-Windows.bat`.

The installer asks whether to create:

- a Desktop shortcut
- a Start Menu entry
- launch the program when installation finishes

The shortcut uses `assets/branding/hamradio-pi-ultimate.ico`.

## Raspberry Pi

Run `./install.sh` or use the anonymous public installer.

The installer creates:

- the `hamradio-pi-ultimate` command
- a **Ham Radio** menu category
- a HamRadio-Pi Ultimate menu entry
- an optional Desktop shortcut

Start-at-login can be enabled later under Preferences.
