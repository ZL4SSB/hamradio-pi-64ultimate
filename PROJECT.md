# HamRadio-Pi Ultimate Project

## Mission

Transform a standard Raspberry Pi OS installation into a complete amateur radio workstation with minimal Linux knowledge.

## Design principles

1. Never require Linux knowledge where plain language will do.
2. Detect hardware and software wherever practical.
3. Explain errors clearly and suggest a solution.
4. Check storage before large installations.
5. Keep essential features free.
6. Put amateur radio tools under one dedicated Ham Radio menu.
7. Preserve user configuration during updates.
8. Never force full-screen operation.
9. Cooperate with other ham-radio applications running on the desktop.
10. Build stations and activities, not merely package lists.

## Editions

- **Community Edition**: complete free core functionality.
- **DX Edition**: optional supporter convenience features only.

## Mascot

**Yobbo the Kiwi** — headset, hand microphone, and compact portable HF transceiver in a backpack.
